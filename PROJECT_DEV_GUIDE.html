Quy trình thêm CSS mới

  1. Bước 1 — Tạo/copy file: src/styles/layout.css (tự viết) hoặc public/vendor/swiper/swiper.min.css (thư viện)
  2. Bước 2 — Mở nuxt.config.js (gốc project) → tìm css: ['@/styles/main.css'] → thêm dòng mới vào mảng
  3. Bước 3 — Restart npm run dev → F12 → Network → filter CSS → thấy file load = OK

Quy trình thêm thư viện JS (file tải về)

  1. Copy ten.min.js → public/vendor/ten/ten.min.js
  2. Mở nuxt.config.js → css: [...] thêm CSS (nếu có)
  3. Cùng file nuxt.config.js → app.head.script: [...] thêm JS
  4. Trong component: onMounted(() => { new Swiper(...) })
  5. Restart npm run dev → F12 → Network → JS status 200
