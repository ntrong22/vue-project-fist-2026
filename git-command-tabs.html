Git & NPM Command Cheatsheet

Tổng hợp các lệnh thường dùng cho project Vue/Vite này: clone, sửa code, push GitHub, cài thư viện, build và deploy IIS.

Tổng quan luồng làm việc

Dùng Git để quản lý source code, dùng npm để cài thư viện/build, dùng thư mục dist để deploy IIS.

Code source

Push lên GitHub: src, public, package.json, package-lock.json, config.

Không push

node_modules, dist, .env đã được chặn trong .gitignore.

Deploy

Build xong copy nội dung trong dist lên IIS.

Kiểm tra nhanh project

bash 
git status
npm install
npm run build

Push project lên GitHub lần đầu

Dùng khi thư mục local chưa có Git hoặc chưa gắn remote GitHub.

Repo ví dụ: https://github.com/ntrong22/vue-project-fist-2026
bash 
cd G:\VUEJS-NEW\pro-vue-01
git init
git branch -M main
git add .
git commit -m "Initial Vue news project"
git remote add origin https://github.com/ntrong22/vue-project-fist-2026.git
git push -u origin main

Kiểm tra trước khi push

bash 
git status
git check-ignore -v node_modules dist .env

Sửa code xong rồi push

Dùng cho các lần làm việc hằng ngày sau khi repo đã kết nối GitHub.

bash 
git status
npm run build
git add .
git commit -m "Update SEO and deploy config"
git push

Commit message ví dụ

bash 
git commit -m "Improve SEO metadata and sitemap config"
git commit -m "Fix contact form layout"
git commit -m "Update API config example"

Lần đầu lấy code từ GitHub về máy

Dùng khi máy chưa có project.

bash 
git clone https://github.com/ntrong22/vue-project-fist-2026.git
cd vue-project-fist-2026
npm install
copy .env.example .env
npm run dev

PowerShell copy file .env

powershell 
Copy-Item .env.example .env

Đồng bộ code với GitHub

Dùng để lấy code mới hoặc xử lý repo đã có commit sẵn.

Lấy code mới nhất

bash 
git pull

Repo GitHub đã có commit sẵn

bash 
git pull origin main --allow-unrelated-histories
git push -u origin main

Xem remote hiện tại

bash 
git remote -v

Làm việc với branch

Branch giúp tách việc sửa code khỏi nhánh chính.

bash 
git branch
git checkout -b seo-update
git add .
git commit -m "Improve SEO setup"
git push -u origin seo-update

Quay về main

bash 
git checkout main
git pull

NPM và thư viện

Dùng để cài package, cập nhật package và build project.

Cài và chạy project

bash 
npm install
npm run dev
npm run build
npm run preview

Quản lý thư viện

bash 
npm install
npm update
npm install ten-thu-vien
npm uninstall ten-thu-vien

Cài đúng theo package-lock

bash 
npm ci

Build và deploy IIS

Project Vue/Vite chỉ cần deploy thư mục dist.

Build production

bash 
npm install
npm run build

Biến môi trường production

env 
VITE_APP_URL=https://domain-cua-ban.com
VITE_API_BASE_URL=https://domain-cua-ban.com/api
VITE_USE_MOCK_API=false
Sau khi build, copy nội dung bên trong thư mục dist lên IIS. Không copy node_modules, src, package.json, .env.

Lỗi thường gặp

Một số lệnh xử lý tình huống hay gặp khi dùng Git/NPM.

Chưa cấu hình Git user

bash 
git config --global user.name "ntrong22"
git config --global user.email "email-cua-ban@example.com"

Remote origin đã tồn tại

bash 
git remote -v
git remote set-url origin https://github.com/ntrong22/vue-project-fist-2026.git

Xóa cache npm khi cài bị lỗi

bash 
npm cache verify
npm install
