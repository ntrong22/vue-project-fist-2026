Git & NPM Command Cheatsheet

Tổng hợp các lệnh thường dùng cho project Vue/Vite này: clone, sửa code, push GitHub, cài thư viện, build và deploy IIS.

Tổng quan luồng làm việc

Dùng Git để quản lý source code, dùng npm để cài thư viện/build, dùng thư mục dist để deploy IIS.

Code source

Push lên GitHub: src, public, package.json, package-lock.json, config.

Không push

node_modules, dist, .env đã được chặn trong .gitignore.

Deploy

Build xong copy nội dung trong dist lên IIS.

Kiểm tra nhanh project

bash 
git status
npm install
npm run build

Push project lên GitHub lần đầu

Dùng khi thư mục local chưa có Git hoặc chưa gắn remote GitHub.

Repo ví dụ: https://github.com/ntrong22/vue-project-fist-2026
bash 
cd G:\VUEJS-NEW\pro-vue-01
git init
git branch -M main
git add .
git commit -m "Initial Vue news project"
git remote add origin https://github.com/ntrong22/vue-project-fist-2026.git
git push -u origin main

Kiểm tra trước khi push

bash 
git status
git check-ignore -v node_modules dist .env

Sửa code xong rồi push

Dùng cho các lần làm việc hằng ngày sau khi repo đã kết nối GitHub.

bash 
git status
npm run build
git add .
git commit -m "Update SEO and deploy config"
git push

Commit message ví dụ

bash 
git commit -m "Improve SEO metadata and sitemap config"
git commit -m "Fix contact form layout"
git commit -m "Update API config example"

Push một số file nhất định

Git không push từng file trực tiếp. Git push các commit. Muốn chỉ đẩy vài file thì chỉ đưa vài file đó vào commit bằng git add file.

Các thay đổi chưa được git add sẽ nằm lại local, không đi theo commit này.

Flow chuẩn: chỉ commit vài file

bash 
git status --short
git add src/pages/ContactView.vue public/web.config git-command-tabs.html
git diff --staged --name-only
git commit -m "Update contact page and Git cheatsheet"
git push

Chỉ thêm một file

bash 
git add README.md
git commit -m "Update README"
git push

File có dấu cách trong tên

bash 
git add "docs/Git Command Note.html"
git commit -m "Add Git command note"
git push

Thêm cả thư mục hoặc một nhóm file

bash 
git add src/pages/
git add src/components/**/*.vue
git add "*.md"

Chỉ lấy một phần thay đổi trong cùng một file

Dùng khi một file có nhiều đoạn sửa, nhưng chỉ muốn commit một vài đoạn.

bash 
git add -p src/pages/ContactView.vue
git diff --staged
git commit -m "Update selected contact changes"
git push

Gỡ file khỏi commit đang chuẩn bị

bash 
git restore --staged src/pages/ContactView.vue
git status --short

Chỉ commit file đã xóa

bash 
git rm old-file.html
git commit -m "Remove old HTML note"
git push

Kiểm tra chắc chắn trước khi commit

bash 
git diff --staged --name-only
git diff --staged
npm run build
  • Không dùng git add . nếu chỉ muốn push vài file.
  • Không commit .env, node_modules, dist.
  • git commit chỉ lấy file đang staged.
  • git push đẩy commit lên GitHub.

Xem thay đổi trước khi commit

Tab này dùng để kiểm tra mình đã sửa gì, file nào đang staged, và lịch sử commit.

Xem trạng thái file

bash 
git status
git status --short

Xem nội dung thay đổi chưa staged

bash 
git diff
git diff src/pages/ContactView.vue
git diff --name-only

Xem nội dung đã staged

bash 
git diff --staged
git diff --staged --name-only
git diff --cached

Xem lịch sử commit

bash 
git log --oneline -10
git log --oneline --graph --decorate --all -15
git show --stat
git show --name-only HEAD

Kiểm tra file có bị .gitignore chặn không

bash 
git check-ignore -v node_modules dist .env
git ls-files
git ls-files package-lock.json

Đọc ký hiệu trong git status --short

text 
?? file-moi.html       file mới, chưa được Git theo dõi
 M src/App.vue         file đã sửa, chưa staged
M  src/App.vue         file đã staged
 D old-file.html       file bị xóa
A  new-file.html       file mới đã staged

Đồng bộ code với GitHub

Dùng để lấy code mới hoặc xử lý repo đã có commit sẵn.

Lấy code mới nhất

bash 
git pull

Repo GitHub đã có commit sẵn

bash 
git pull origin main --allow-unrelated-histories
git push -u origin main

Xem remote hiện tại

bash 
git remote -v

Làm việc với branch

Branch giúp tách việc sửa code khỏi nhánh chính.

bash 
git branch
git checkout -b seo-update
git add .
git commit -m "Improve SEO setup"
git push -u origin seo-update

Quay về main

bash 
git checkout main
git pull

Đổi branch nhanh

bash 
git switch main
git switch -c fix-contact-form
git push -u origin fix-contact-form

Xóa branch đã làm xong

bash 
git branch -d fix-contact-form
git push origin --delete fix-contact-form

Undo và stash

Dùng khi add nhầm file, sửa nhầm file, muốn đổi commit cuối hoặc muốn cất tạm code trước khi pull.

Gỡ file khỏi staged nhưng giữ nội dung đã sửa

bash 
git restore --staged src/pages/ContactView.vue
git restore --staged .
git status --short

Bỏ thay đổi local của một file

bash 
git restore src/pages/ContactView.vue
git restore README.md

Sửa commit cuối khi chưa push

bash 
git commit --amend -m "Update Git and NPM cheatsheet"

# Thêm file quên vào commit cuối
git add git-command-tabs.html
git commit --amend --no-edit

Tạm cất thay đổi để pull hoặc đổi branch

bash 
git stash push -m "dang sua contact page"
git pull
git stash pop

Xem và quản lý stash

bash 
git stash list
git stash show -p stash@{0}
git stash apply stash@{0}
git stash drop stash@{0}

Hoàn tác commit đã push bằng commit mới

bash 
git log --oneline -5
git revert HEAD
git push
  • git restore --staged chỉ bỏ khỏi vùng commit, không mất code.
  • git restore file sẽ bỏ thay đổi của file đó, nên kiểm tra kỹ bằng git diff.
  • git revert phù hợp khi commit đã push lên GitHub.

NPM và thư viện

Dùng để cài package, cập nhật package và build project.

Cài và chạy project

bash 
npm install
npm run dev
npm run build
npm run preview

Quản lý thư viện

bash 
npm install
npm update
npm install ten-thu-vien
npm uninstall ten-thu-vien

Cài đúng theo package-lock

bash 
npm ci

Xem package đang dùng

bash 
npm list --depth=0
npm outdated
npm explain vite
npm explain vue

Cài package vào đúng nhóm

bash 
# Thư viện chạy trong app
npm install axios

# Thư viện chỉ dùng khi dev/build
npm install -D vite tailwindcss

Kiểm tra bảo mật và dọn cài đặt

powershell 
npm audit
npm audit fix
npm cache verify

# Nếu node_modules bị lỗi, xóa rồi cài lại
Remove-Item -Recurse -Force node_modules
npm install

Build và deploy IIS

Project Vue/Vite chỉ cần deploy thư mục dist.

Build production

bash 
npm install
npm run build

Biến môi trường production

env 
VITE_APP_URL=https://domain-cua-ban.com
VITE_API_BASE_URL=https://domain-cua-ban.com/api
VITE_USE_MOCK_API=false

Xem nội dung build ra

powershell 
Get-ChildItem dist
Get-ChildItem dist -Recurse | Select-Object FullName

Preview bản production trước khi copy lên IIS

bash 
npm run build
npm run preview

Checklist trước khi deploy

  • Chạy npm run build thành công.
  • Kiểm tra dist/index.html, dist/robots.txt, dist/sitemap.xml.
  • Copy nội dung bên trong dist, không copy nguyên thư mục dist nếu IIS đang trỏ vào web root.
  • Đảm bảo có web.config để F5 route con không bị 404.
Sau khi build, copy nội dung bên trong thư mục dist lên IIS. Không copy node_modules, src, package.json, .env.

Conflict và đăng nhập GitHub

Dùng khi pull/push bị lỗi do code trên GitHub mới hơn local, merge conflict, sai remote hoặc GitHub yêu cầu đăng nhập.

Push bị non-fast-forward

bash 
git status
git pull --rebase origin main
npm run build
git push

Khi có conflict

bash 
git status

# Mở file báo conflict, tìm các đoạn:
# <<<<<<< HEAD
# =======
# >>>>>>> branch-hoac-commit

git add file-bi-conflict.vue
git rebase --continue
git push

Không muốn tiếp tục merge/rebase nữa

bash 
git merge --abort
git rebase --abort
git status

Kiểm tra và sửa remote

bash 
git remote -v
git remote set-url origin https://github.com/ntrong22/vue-project-fist-2026.git
git branch -vv

Gắn upstream khi push lần đầu của branch

bash 
git push -u origin main
git push -u origin ten-branch-cua-ban

GitHub không nhận mật khẩu

text 
GitHub không dùng password tài khoản để push qua HTTPS.
Khi Git hỏi mật khẩu, dùng Personal Access Token.

Username: tên GitHub
Password: token GitHub
  • Trước khi pull, nếu đang sửa dở: commit hoặc stash trước.
  • Sau khi resolve conflict: chạy npm run build trước khi push.
  • Không dùng force push nếu chưa chắc chắn vì có thể ghi đè lịch sử làm việc của người khác.

bash

Remote origin đã tồn tại

bash 
git remote -v

Xóa cache npm khi cài bị lỗi

bash 
npm cache verify
npm install

bash

git add package.json package-lock.json
